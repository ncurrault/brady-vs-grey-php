Project by Nicholas Currault. The source code is available on GitHub.

To conserve processor and network resources, this app should only update four times per day.

This web app is available in two views:

  • Improved
  • Classic

Updates

On 2020-10-01, a "behind the scenes" update made the app much cheaper to host, but the first new request of each day will take a while to load. If you get a blank page, try again in a few minutes.
